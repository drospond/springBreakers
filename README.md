# Spring Breakers

This is a travel app targeting college students looking to find cheap places to stay and good restaurants to eat at on their spring break vacations. 

## Technologies
* HTML5
* Bulma CSS
* JavaScript
* jQuery
* 3rd party APIs
    * Zomato
    * Trip Advisor
